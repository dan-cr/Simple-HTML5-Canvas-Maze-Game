Old Worlde Phunne Maze Game

Instructions:

  1. Enter a number of cells or select a difficulty.
  2. Click the generate button to create the maze.
  3. Navigate Thylanaes the Night Elf through the maze to collect the gold.
  4. Note: Move using the WSAD keys.
